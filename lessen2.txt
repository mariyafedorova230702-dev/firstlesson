1. git init  - Это команда создает конфигурацию гит в текущей папке
	Вводится 1 раз при создании проекта 

2. git remote add origin... - Создаем связь между репозиторием и 
				нашей папкой
	Вводится 1 раз после инициализации 

3. git add [название файла или папки] - добавляем изменения в гит
	. - означет все файли и папки в текущей директории 
     	Вводим каждый раз когда хотим сохранить изменения

4. git commit -m "Коментарий" - Пишем комментарий того что сделали 

5. git push origin master - Отправляет все наши изменения в github
