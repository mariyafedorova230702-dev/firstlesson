1. pwd - Показывает где вы находитесь 
2. mkdir - Создает папку
3. cd Название папки  - Открыть папку 
4. cd .. - Назад
5. ls    - Показывает содержимое папки 
6. rm Название папки - Удаляет файл 
8. rm -d Название папки - Удаляет папку если она пустая
9. rm -rf Название папки файла -  Удаляет папку файл
10. touch Название файла - Создает файл
11. nano Название файла - Открыть редактирование
